# Trabajo Final de VA
Proyecto de visión artificial para detección de vehículos

Proyecto Visión Artificial orientado a la detección de vehículos
en escenarios de tráfico urbano usando YOLOv8 y transfer learning.

## Contenido
- Trabajo_Final.ipynb: notebook principal del proyecto
- Dataset: obtenido desde Kaggle (no incluido)
- Modelo: YOLOv8
- Métricas: Precision, Recall, mAP

## Resultados
El modelo obtiene un buen desempeño en detección vehicular,
especialmente en clases como autos y buses.
